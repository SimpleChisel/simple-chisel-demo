** Installation
To Compile the codes, first install `SimpleChisel` locally

```shell script
git clone git@gitlab.eecs.umich.edu:ablab/simple-chisel.git
cd simple-chisel
git checkout release
sbt publishLocal
```

** Compile Examples
To compile examples, execute the following commands:

```shell script
make
```
